Contributing to Smart Hibernator

Thanks for your interest! Here's how you can help.

## Development Setup
```bash
git clone https://github.com/lizzyman04/smart-hibernator.git
cd smart-hibernator
npm install
npm run dev
```

## Code Style
- TypeScript for all new code
- 2 spaces indentation
- Prettier formatting (config in repo)

## Commit Convention
- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `test:` tests
- `chore:` maintenance
- `refactor:` code change

## Pull Request Process
1. Ensure all tests pass (`npm test`)
2. Build successfully (`npm run build`)
3. Update documentation if needed
4. Request review

## Reporting Issues
- Use the GitHub issue tracker
- Include steps to reproduce
- Attach extension logs if possible
