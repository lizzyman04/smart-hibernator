# Phase 2 — UI Design Contract

> Visual and interaction contract for Phase 2: UI & Rich Previews.
> Generated by gsd-ui-researcher. Verified by gsd-ui-checker.
> Extends Phase 1 contract without contradicting it.

---

## Scope

This contract covers four UI surfaces delivered in Phase 2:

1. **Popup (redesigned)** — `src/popup/App.tsx` — 320px fixed-width; now a hibernated-tab manager with scrollable list, per-tab Wake Tab buttons, and a Dashboard footer link. Phase 1 global toggle and protect toggle retained.
2. **Dashboard page** — `src/dashboard/index.html` — full-page extension page with Stats tab and Settings tab.
3. **Thumbnail display** — WebP thumbnail shown in popup list rows; fallback card shown when no capture exists.
4. **Settings controls** — timeout slider (5–240 min) and domain whitelist chips, rendered inside the Dashboard Settings tab.

Deferred from Phase 2 (do not implement): Tab Group protection, Chrome Side Panel, per-tab RAM measurement, URL-keyed thumbnail reuse across sessions.

## Spacing Scale

Identical to Phase 1 — extended to cover dashboard layout tokens:

| Token | Value | Tailwind Class | Usage |
|-------|-------|----------------|-------|
| xs | 4px | `p-1` / `gap-1` | Icon gaps, tight inline padding, thumbnail badge offsets |
| sm | 8px | `p-2` / `gap-2` | List row internal gaps, chip internal padding |
| md | 16px | `p-4` / `gap-4` | Default element spacing, popup section padding |
| lg | 24px | `p-6` / `gap-6` | Popup outer padding, dashboard section padding |
| xl | 32px | `p-8` / `gap-8` | Dashboard card internal padding, chart container padding |
| 2xl | 48px | `p-12` | Major dashboard section breaks |
| 3xl | 64px | `p-16` | Dashboard hero metric vertical breathing room |

**Popup outer padding:** `p-6` (24px all sides) — unchanged from Phase 1.

**Dashboard outer padding:** `p-8` (32px all sides) — wider than popup to use full-page real estate.

**List row height:** `min-h-16` (64px) — accommodates thumbnail (48px tall) + 8px top/bottom padding.

**Thumbnail dimensions:** 80×48px rendered in popup list rows (maintains 5:3 aspect ratio, fits within 64px row height with 8px vertical padding each side).

**Exceptions:**
- Touch target minimum: 44px tall for all interactive controls — `min-h-11` on row wrappers. Source: Phase 1 established.
- Slider thumb: 20px diameter (`h-5 w-5`) — standard shadcn Slider default.
- Chip (domain whitelist badge) height: 28px (`h-7`) with `px-2` horizontal padding.
